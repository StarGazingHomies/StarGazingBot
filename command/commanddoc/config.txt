Configures SERVER settings. Only usable by server owners and bot owner.

**Usage:**
`{prefix}config [page]` shows settings
`{prefix}config <setting>` shows help  message for <setting>
`{prefix}config <setting> <value>` sets <setting> as <value>

**Example:**
`{prefix}config 2`
`{prefix}config apply False`

**Aliases:**
`{prefix}config prefix <value>` = `{prefix}prefix <value>` or `{prefix}pfx <value>`
